# 05 · 开发路线图与里程碑

按「先骨架、再安全、后扩展」推进。每个里程碑都应是**可运行、可演示**的增量。

## 里程碑 M0 · 工程脚手架（1 周）
- 建 Xcode/SPM 工程，分层目录（App / Domain / Infrastructure / DesignSystem / Features / Helper / Shared / Tests）。
- 通用二进制、最低 macOS 13、Swift 6 严格并发开启。
- DesignSystem 基础：Design Tokens、颜色、字体、主窗口 + 侧边栏空壳、深浅色。
- CI：构建 + 单测 + SwiftLint/格式化。
- **验收**：能跑出带侧边栏与毛玻璃的空窗口，深浅色正常。

## 里程碑 M1 · 权限与特权助手地基（1–2 周）
- `PermissionsManager`：检测/引导完全磁盘访问（FDA）；引导页 `XPermissionGate`。
- `SMAppService` 注册特权助手；定义 XPC 协议（`HelperProtocol`）；双向代码签名校验。
- 助手暴露最小白名单接口（先实现「校验后删除路径列表」+「读容量」）。
- **验收**：首启引导授予 FDA + 安装助手；主应用经 XPC 让助手安全删除一个测试文件。

## 里程碑 M2 · 安全引擎 + 扫描/清理内核（2 周）
- `SafetyEngine`（保护清单、路径规范化、越界防护、废纸篓优先、撤销）。
- `ScanCoordinator` + `CleaningEngine`（事务、日志、回滚）。
- 统一 `ScanResult` 模型 + 统一结果列表 UI（`XResultList`/`XItemRow`/`XActionBar`）。
- **安全回归测试**先行（删保护路径必被拒）。
- **验收**：跑通「扫描→预览→清理→撤销」闭环（先用一个简单模块验证内核）。

## 里程碑 M3 · 首批核心模块（3–4 周）
- **系统垃圾**（数据驱动定义库 + 基线规则 + 安全分级）。
- **大文件与旧文件**。
- **废纸篓清空**。
- **智能扫描**：聚合上述模块，做首页大圆环 + 一键清理 + 完成庆祝动效。
- **验收**：能真实释放空间，全程预览可控，动效到位。

## 里程碑 M4 · 空间透镜（2–3 周）
- 高性能全盘扫描（`getattrlistbulk` 封装 + actor 聚合 + 有界并行）。
- Treemap 自绘 + 钻取/面包屑/悬停；Sunburst 切换。
- 右键「Finder 显示 / 移废纸篓」。
- **验收**：对整块 SSD 数十秒出图，交互流畅不卡。

## 里程碑 M5 · 应用与文件模块（3 周）
- **卸载器** + 残留扫描。
- **重复文件**（三阶段 + 克隆识别）。
- **开发者垃圾**（差异化卖点）。
- **验收**：彻底卸载某 App 含关联文件；正确识别重复且不误报克隆。

## 里程碑 M6 · 性能/安全/常驻（2–3 周）
- **优化**（登录项/启动项/高耗进程）。
- **维护**（脚本/Spotlight/DNS/快照，经助手）。
- **隐私**（浏览器数据清理）。
- **菜单栏监控**（MenuBarExtra 实时指标 + 快捷释放内存）。
- **验收**：菜单栏常驻稳定低耗；维护操作经助手安全执行。

## 里程碑 M7 · 打磨与商业化（2–3 周）
- 全面动效打磨、空态/错误/撤销文案、可访问性、本地化（中/英）。
- Sparkle 自动更新 + 在线定义库更新 + 签名。
- 公证 + DMG 打包；FastSpring/Paddle 许可与试用。
- 隐私政策、可关遥测。
- **验收**：可对外分发的 1.0，Gatekeeper 放行，能更新、能购买。

## 里程碑 M8（Phase 2 · 可选）
- 威胁/恶意软件移除（特征库 + 后台）。
- 应用更新器、相似图片查重（pHash）、外置盘管理、计划任务（定时自动清理）。

## 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| **误删用户数据** | 致命，毁口碑 | SafetyEngine + 废纸篓优先 + 安全回归测试为发版红线 + 默认只勾 safe |
| 非沙盒被苹果政策变动影响 | 分发受阻 | 紧跟 Developer ID/公证政策；保留 App Store 阉割版预案 |
| 特权助手安全漏洞 | 提权风险 | 助手只接白名单参数化操作 + 双向签名校验 + 助手侧独立复校 |
| APFS 行为误导用户（快照/克隆/可清除） | 信任受损 | 正确口径展示 + 文案解释 + 克隆识别 |
| 扫描性能/内存 | 体验差 | getattrlistbulk + 有界并行 + 小文件聚合 + 性能基准防回归 |
| 定义库过期（新系统/新 App） | 清理能力退化 | 在线可更新定义库 + 离线兜底 |

## 团队与工时（粗估）
- 1 名资深 macOS/Swift 工程师为主力，约 **4–5 个月**可达 1.0。
- 若 1 设计 + 2 工程并行，可压缩到 **2.5–3 个月**。
- 用 Codex/Claude 辅助可显著加速脚手架、模块样板、测试编写（见 [CODEX_PROMPT.md](../CODEX_PROMPT.md)）。
