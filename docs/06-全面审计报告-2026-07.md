# Xico 全面产品审计报告（2026-07）

> 审计方式：多智能体并行审计，7 个代码维度（安全红线 / 特权助手 XPC / 扫描器 / UX 产品 / 分发商业化 / 测试 / 工程质量）+ 1 个 CleanMyMac 竞品研究。
> 共产出 88 条发现；全部 P0/P1 代码级发现经独立"反驳式"核实：**28 条确认，1 条被驳回降级**。
> 测试基线：`swift test` 63 个测试全部通过（2 个跳过）。

---

## 一、总体结论

**Xico 不是半成品——完成度约 70%，架构是同类独立项目中的上游水准。** 12 个侧栏模块全部有真实实现，安全红线架构（共享 SafetyRules + helper 侧 openat/O_NOFOLLOW fd 锚定删除）达到商用水准，签名许可证/规则库更新链路骨架正确，视觉品质约 80 分。

"半成品感"来自三类问题，按危险程度排序：

1. **安全红线有一个致命缺口**：保护清单漏掉了当代 Mac 上最大的数据损失面——iCloud Drive、CloudStorage、邮件、信息、照片图库包、Application Support 全部被红线放行（已编译实测确认）。云同步目录被删会同步到云端和所有设备，废纸篓救不回来。这是"产品死亡"级事故的入口。
2. **商业闭环断裂**：没有支付链路（P0，收不到钱）、没有 Sparkle 自动更新（发出去就失联）、许可证一条 `defaults write` 就能破解、CI 假绿（测试失败照样绿灯，安全门禁形同虚设）。
3. **用户第一天就会撞上的闭环硬伤**：FDA 授权后横幅不消失、结果组只渲染 80 项但隐藏项仍会被删、撤销入口一闪而过、试用到期被瞬间弹去设置页。

**距离"敢收钱发布"约 2-4 周专注工作；距离对 CleanMyMac 的功能对标约 2-3 个月。**

## 三、对标 CleanMyMac：差距与打法

### 现状对比

CleanMyMac（MacPaw，现 5.x）已演化为"全能 Mac 护理套件"：Smart Care 一键五合一、Moonlock 威胁引擎、云存储清理、AI 智能洞察、菜单栏健康监控。定价：订阅 $39.95/年起，买断 $119.95，Setapp $9.95/月。

**Xico 已覆盖其约 70% 高频功能**，且有一项对手没有的能力：**清理历史 + 可撤销**——直接命中"清理软件会不会删坏我系统"这一最大购买顾虑，应提到品牌层面。

主要功能缺口（按购买决策重要性）：

| 缺口 | 重要性 | 说明 |
|---|---|---|
| 可用的恶意软件移除 | 核心卖点 | "安全"是购买决策第一关键词；对手病毒库更新慢、无 AV-TEST 认证，对齐门槛没那么高 |
| Sparkle 自动更新 | 发布刚需 | 1-2 天工作量，没有它一切迭代无法触达用户 |
| 相似图片/截图清理 | 重要 | 感知价值最高，"一键找回 20GB"是最好的营销素材，Vision 框架可本地实现 |
| 应用更新器 | 重要 | Sparkle appcast 探测 + Homebrew cask 数据源 |
| 菜单栏性能图表 | 重要 | 可复用现有 LiveMetrics，提升日活 |
| 邮件附件清理 / Shredder | 锦上添花 | 低成本清单项功能 |
| 云存储清理 | 建议不做 | 成本最高且与"纯本地隐私"定位冲突——把"我们不碰你的云端数据"写成卖点比实现它更划算 |

### 定位建议：不要正面打"全能护理"，叠加四层差异化

1. **一次性买断、永不订阅**：Reddit/评测区对 CleanMyMac 订阅制的反感是现成流量。建议 $34.99 买断（国区 ¥168）含 1 年更新，次年 $19.99 可选升级包（Sketch 模式）；早鸟 $24.99。
2. **隐私与本地优先**：零遥测、许可证离线 Ed25519 验证、规则库只下载签名文件不上传数据——官网附 Little Snitch 网络流量审计截图，对开发者/隐私用户是决定性卖点。
3. **开发者垃圾细分市场**：Xcode/node_modules/Docker/Homebrew/gradle/cargo——数据驱动规则库天然适合快速扩展，目标用户聚集在 HN/V2EX/少数派，获客成本最低。
4. **中文市场**：CleanMyMac 中国区支付不便、无本土运营。简体中文原生 + 微信/支付宝支付 + 少数派/B 站内容营销，"国人开发者的 CleanMyMac 平替、买断制"是被 iShot/Downie 验证过的叙事。

渠道：官网直销走 Paddle 或 Lemon Squeezy（代扣全球税 + 支持支付宝）；中国区可谈数码荔枝代理；6-12 个月后评估 Setapp；不上 Mac App Store 主版本（root 助手过不了沙盒）。
