## 1. 证据等级和状态词

后续接手人必须使用以下状态，禁止混用：

| 状态 | 含义 | 能否算完成 |
|---|---|---|
| verified | 当前精确 revision 有代码、测试、构建、审查和所需 artifact | 可以 |
| committed / needs revalidation | 代码已提交，但当前审查包、最新回归或现场证据缺失 | 不能作为最终验收 |
| implemented-uncommitted | 工作区有实现，尚未提交或尚未完成全门 | 不能 |
| historical evidence | 旧 revision 曾通过，当前 HEAD 已变化 | 不能作为最终发布签字 |
| lost-uncommitted | 会话中曾实现或测试，但未提交，工作区已丢失 | 不能；必须重建 |
| planned | 有规格或计划，未实现 | 不能 |
| open | 未开始或未形成可执行闭环 | 不能 |
| external/open | 依赖真实证书、设备、服务、用户或竞品实验室 | 不能 |
| superseded | 有正式决策记录证明被新方案取代 | 只有决策完整时可不实现旧项 |

证据优先级：

1. 当前文件系统、Git commit/tree、测试输出、构建和安装包检查。
2. 当前计划、progress、固定 review package。
3. 历史交接与会话记录。
4. 旧“全部完成”“全球顶尖”文字声明。

文字不能覆盖代码事实；focused test 不能覆盖 full suite；build 不能覆盖安装；codesign valid 不能覆盖 notarization；分支提交不能覆盖当前已安装 App。

要达到比 CleanMyMac 更高级，视觉不是堆渐变、粒子、发光和玻璃，而是同时满足：

1. 第一眼可理解。
2. 数据真实，不用动画掩盖未知。
3. 高风险操作的选择、确认、结果和恢复清楚。
4. 跨页面一致。
5. 轻/深色和四主题稳定。
6. 德/俄/日长文本不破版。
7. VoiceOver、键盘、Reduce Motion/Transparency 完整。
8. 动画结束后 CPU/GPU 回落。
9. 当前安装包和截图与代码一致。
10. 同设备、同任务、盲评数据优于竞品。

### 7.2 当前可见进展

Smart Scan 分支已经建立：

- 更稳定的页面骨架。
- Today Bar、Scan Instrument、Insight Stack。
- 六类真实进度。
- All Tools、Recent Tools、推荐和发现 rail。
- 有限时长 motion。
- 更诚实的取消和非可信状态。

这是一条正确的高端方向，但尚未完成 Review、Executing、Terminal、AX、visual matrix、performance、安装和 live gate。

### 7.3 飞跃式 UI 的推荐实施顺序

#### 第一层：先让用户能看到

1. 恢复并重审 Smart Scan Task 6。
2. 重建 Task 7 Review/Executing。
3. 完成 Task 8 Terminal。
4. 完成 Task 9 AX/11 语言/窗口。
5. 完成 Task 10 视觉矩阵和性能。
6. 完成 Task 11 安全安装并让用户现场确认。

#### 第二层：统一全产品 shell

1. Root、Sidebar、All Tools、Command Palette。
2. 标题、Hero、列表、卡片、底栏和状态模式。
3. 统一 DesignSystem、XMotion、focus、keyboard。
4. 删除各页面独有的裸色、裸字体、裸圆角和散落 spring。

#### 第三层：优先升级用户最能感知的页面

1. Smart Scan。
2. Dashboard。
3. Uninstaller。
4. Space Lens。
5. Duplicates/Similar Images。
6. Maintenance/Optimization。
7. Monitor/Hardware/Network。
8. Settings/Pricing/Onboarding/Menu Panels。

#### 第四层：用证据证明“超过”

- 两位独立视觉 reviewer，使用同一 rubric，取较低分。
- 差异 ≥5 分时第三人裁决。
- 10 名目标用户做首次任务理解和可发现性测试。
- 与当前 CleanMyMac 在同一台 Mac、同一数据集、同一任务盲测。
- 记录完成时间、误操作率、理解率、撤销率、FMC、输入 P95、settled GPU。

没有这些证据，“比 CleanMyMac 更高级”只能是设计意图，不是验收结论。
