# 03 · UI/UX 设计系统

> 目标：达到甚至超越 CleanMyMac X 的「高级、清爽、愉悦」质感。设计哲学 = **克制的精致** + **满足感的动效** + **绝对的清晰**。

## 1. 设计原则

1. **清晰优先于花哨**：信息层级一眼明了，删除/危险操作绝不含糊。
2. **大留白、强主次**：每屏只有一个主行动（一个大按钮 / 一个主可视化）。
3. **原生为骨，品牌为皮**：遵循 macOS HIG 的交互逻辑，在视觉皮肤上做品牌化（毛玻璃、渐变、自定义插画）。
4. **动效服务于反馈**：进度、完成、释放空间用动效传达「确实做了事」的满足感，而非装饰。
5. **可访问 + 可关动效**：尊重 Reduce Motion / Increase Contrast / 动态字体 / VoiceOver。

│               │                                              │
│   侧边栏       │              主画布 (Hero Canvas)             │

│  · 系统垃圾    │   · 空间透镜：全屏 treemap / 旭日图           │
│  · 空间透镜    │                                              │
│  · 大文件      │                                              │

│               │                                              │

- 侧边栏使用 `NSVisualEffectView`/SwiftUI `.background(.ultraThinMaterial)` 的毛玻璃半透明。
- 整窗体启用 `titlebarAppearsTransparent` + 全尺寸内容视图，营造一体感。
- 窗口最小尺寸约 1000×680，自适应放大。

## 3. 色彩系统（Design Tokens）

采用语义化 Token，全部支持深/浅色自动切换。以下为建议值（可微调）：

### 品牌色
| Token | 浅色 | 深色 | 用途 |
|-------|------|------|------|
| `brand/primary` | `#3B6FF6` 电光蓝 | `#5B8DFF` | 主按钮、强调、进度 |
| `brand/primaryGradient` | `#3B6FF6 → #6A5BFF` | 同向更亮 | 大圆环、Hero |
| `accent/success` | `#27C28A` 翠绿 | `#34D399` | 清理完成、健康 |
| `accent/warning` | `#F5A623` 琥珀 | `#FBBF24` | 有风险项 |
| `accent/danger` | `#FF4D5E` 珊瑚红 | `#FF6B7A` | 危险/磁盘满 |

### 中性色 / 背景
| Token | 浅色 | 深色 |
|-------|------|------|
| `bg/canvas` | `#F5F6FA` | `#1C1C1E` |
| `bg/surface` (卡片) | `#FFFFFF` | `#2C2C2E` |
| `bg/elevated` | `#FFFFFF` + 阴影 | `#3A3A3C` |
| `text/primary` | `#1A1A2E` | `#F2F2F7` |
| `text/secondary` | `#6B6B7B` | `#A0A0A8` |
| `border/subtle` | `#E5E6EC` | `#3A3A3C` |

> 深色模式优先使用系统 Material（`.ultraThinMaterial` 等），让背景与桌面壁纸产生真实毛玻璃，是「高级感」的关键。

## 4. 排版

- **字体**：系统 **SF Pro**（标题用 SF Pro Display，正文 SF Pro Text）。数字（容量/进度）用 **等宽数字** (`.monospacedDigit()`)，跳动时不抖。
- **字号阶梯**：
  - Hero 数字 48–64pt（如「释放了 12.4 GB」）
  - Title 22pt / Headline 17pt 半粗
  - Body 13–15pt / Caption 11–12pt
- 行高宽松，字重对比强（标题 semibold/bold，正文 regular）。

## 5. 核心组件库（DesignSystem package）

| 组件 | 说明 |
|------|------|
| `XSidebar` | 分组导航，选中态有柔和高亮 + 图标着色 |
| `XSummaryCard` | 顶部汇总卡（可释放空间 / 问题数），大数字 + 进度 |
| `XScanRing` | 智能扫描中央大圆环（进度 + 粒子 + 渐变描边） |
| `XResultList` | 统一结果列表：分组、复选框、大小、展开详情、风险标签 |
| `XItemRow` | 单行：图标 + 名称 + 路径(次要) + 大小 + 状态徽标 |
| `XActionBar` | 底部固定操作条：已选统计 + 主按钮 |
| `XDiskBar` | 磁盘空间分段条（系统/应用/文档/可清除/可用） |
| `XTreemap` / `XSunburst` | 空间透镜可视化（自绘） |
| `XEmptyState` | 空态 / 全部干净的庆祝态 |
| `XPermissionGate` | 权限缺失引导卡 |
| `XToast` / `XUndoBar` | 操作反馈 + 撤销 |
| `XBadge` | 安全/警告/危险 语义徽标 |

所有组件从 Design Tokens 取色取间距，**禁止硬编码颜色/尺寸**。间距用 4pt 基准网格（4/8/12/16/24/32）。圆角统一（卡片 12–16、按钮 8–10、大容器 20）。

## 6. 标志性动效（Signature Motion）

动效是 CleanMyMac 让「清理」变愉悦的灵魂。Xico 的招牌动效：

1. **扫描圆环**：扫描时圆环渐变描边旋转 + 沿环飞散的轻量粒子，中心数字实时滚动（monospacedDigit）。
2. **结果汇聚**：扫描完成，发现项以微小卡片从中心「吸入」到列表，配合轻弹性（spring）。
3. **清理流动**：点「清理」时，被清项目化作光点流向中央「释放」动画，进度环收束。
4. **完成庆祝**：完成页大数字弹出「释放了 X GB」+ 一次克制的礼花/光晕（非廉价撒花）。
5. **磁盘条变化**：清理后磁盘空间条平滑回缩可用区扩大，给出「肉眼可见的成果」。
6. **微交互**：悬停抬升、复选打勾的弹性、按钮按压回弹。

技术：SwiftUI `withAnimation(.spring)`、`matchedGeometryEffect`、`Canvas`/`TimelineView` 做粒子、必要时用 Metal/`CAEmitterLayer` 做高性能粒子。**所有动效在 Reduce Motion 下降级为淡入淡出。**

## 7. 空间透镜可视化（Space Lens）

- 默认 **Treemap（矩形树图）**：面积 = 占用大小，颜色按文件类型/深度。支持点击钻取、面包屑返回、悬停显示大小、右键「在 Finder 显示 / 移到废纸篓」。
- 可切 **Sunburst（旭日图）**：环形层级，更直观看目录嵌套。
- 性能：自绘于 `Canvas` 或 Metal 层；大数据集做 LOD（小于阈值的块合并为「其他」）；增量渲染。
- 交互细腻：钻取有缩放过渡动画，定位到某文件时高亮路径。

## 8. 状态与反馈语言

- **扫描中**：进度 + 已发现统计实时滚动，可随时取消（取消要干净、即时）。
- **空态**：「太棒了，这里很干净 ✨」配插画，而非冷冰冰的空白。
- **危险确认**：删除有风险项前用 sheet 二次确认，列出将删内容与可恢复性说明。
- **错误**：友好可操作的错误提示（如「需要完全磁盘访问权限」→ 一键去设置）。
- **撤销**：清理后顶部出现「已清理 X GB · 撤销」浮条，留几秒窗口。

## 9. 深浅色与主题

- 完整支持系统深/浅色自动切换；可在设置中强制某一种。
- 强调色可让用户选（蓝/紫/绿等品牌色板），增强个性。
- 高对比模式：自动加深边框、降低半透明。

## 10. 图标与插画

- 功能图标统一用 **SF Symbols**（保证与系统一致 + 自动适配字重/颜色），关键品牌处用自定义插画。
- App 图标：现代、有辨识度（建议一个简洁的「光/磁盘/扫描环」意象），多分辨率齐全。
- 可选品牌吉祥物（类似 CleanMyMac 的 Mac 小助手），用于引导/空态/完成页增加亲和力——但保持克制、专业。

## 11. 可访问性清单

- [ ] VoiceOver 标签完整，列表可键盘遍历
- [ ] 全键盘可达（Tab/方向键/回车/空格勾选）
- [ ] 动态字体不破版
- [ ] Reduce Motion 降级动效
- [ ] 对比度 ≥ WCAG AA
- [ ] 颜色不作为唯一信息载体（危险项除颜色外有图标/文字）
