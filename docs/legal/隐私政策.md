# Xico 隐私政策

**生效日期：2026-07-02**

Xico（以下简称「本软件」）是一款在你的 Mac 本地运行的系统清理与磁盘管理工具。我们把隐私作为核心设计原则。

## 一、我们不收集什么

本软件**不收集、不上传、不出售**你的任何个人数据。具体而言：

- **零遥测**：本软件不内置任何分析、埋点或使用统计上报。
- **零云端**：所有扫描、清理、磁盘分析均在你的设备本地完成，文件内容与路径**从不离开你的 Mac**。
- **不读取文件内容用于任何目的**：查重仅在本地计算文件哈希以判断内容是否相同，哈希不上传。

你可以用 Little Snitch 等网络监控工具验证：本软件仅在你**主动**触发以下操作时才发起网络请求。

## 二、有限的网络请求（均由你触发或可关闭）

1. **规则库在线更新**：从我们的服务器下载经 Ed25519 签名的清理规则库。请求仅包含标准 HTTP 头，不含任何设备标识或个人信息。
2. **许可证 / 购买**：点击「购买」会用系统浏览器打开购买页；许可证验证在**本地离线**完成（Ed25519 签名校验），不联网。
3. **检查更新**：点击「检查更新」时读取版本信息（appcast），仅用于比较版本号。

以上请求都不携带可识别你个人身份的信息。

## 三、本地存储的数据

本软件在 `~/Library/Application Support/Xico/` 与系统钥匙串中保存：清理历史（仅统计数字与本地路径映射，用于「撤销」）、试用起始时间、许可证文件。这些数据仅存于你的设备，卸载本软件后可手动删除。

## 四、诊断日志

「设置 › 导出诊断日志」会把本软件最近的运行日志导出为文本文件到你选择的位置，供你在反馈问题时**自愿**提供给我们。本软件不会自动发送任何日志。

## 五、第三方支付

购买通过第三方支付平台（如 Paddle / Lemon Squeezy）完成，你的支付信息由该平台按其隐私政策处理，本软件不接触你的支付卡信息。

## 七、政策变更

如本政策有实质变更，我们会更新本文件的生效日期并在应用内提示。
