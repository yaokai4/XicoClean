# 01 · 产品定义与功能规划

## 1. 产品愿景

Xico 是一款面向 macOS 的**系统清理、磁盘管理与性能优化**桌面应用，目标是给用户「一键让 Mac 重新变快、变干净、腾出空间」的确定性体验，同时保持**高级、清爽、令人愉悦**的视觉与交互质感。

**核心价值主张：**
- **腾空间**：精准找出系统垃圾、大文件、重复文件、应用残留，安全释放磁盘。
- **看得见**：用空间透镜（treemap / 旭日图）把整块 SSD 的占用可视化，让用户一眼看懂「空间去哪了」。
- **更安心**：所有删除都可预览、可白名单、可撤销，绝不误删系统关键文件。
- **够好看**：媲美 CleanMyMac X 的精致动效与界面，让「清理」这件枯燥的事变得有满足感。

## 2. 目标用户

| 用户群 | 痛点 | Xico 解决方案 |
|--------|------|---------------|
| 普通 Mac 用户 | SSD 报满、电脑变卡、不懂哪能删 | 一键智能扫描 + 安全清理 |
| 开发者 | Xcode DerivedData / 模拟器 / node_modules 占满几十 GB | 开发者垃圾专项清理 |
| 设计师 / 创作者 | 大量素材、缓存、重复文件、外置盘混乱 | 大文件 + 重复文件 + 空间透镜 |
| 进阶 / 极客用户 | 想精细控制启动项、维护脚本、内存释放 | 优化 + 维护 + 菜单栏监控 |

## 3. 产品形态

1. **主应用窗口**（Developer ID 分发的原生 App）。
2. **菜单栏常驻小组件**（MenuBarExtra）：实时显示 CPU / 内存 / 磁盘 / 网络 / 电池，支持快速释放内存、快速清理。
3. **特权助手守护进程**（Privileged Helper，root）：执行需要管理员权限的清理与维护。
4. **后台定义更新服务**：清理规则库 / 恶意软件特征库的在线更新。

## 4. 功能模块全景（信息架构）

主窗口采用「左侧侧边栏 + 右侧主画布」布局。模块分为四大组：

### A. 清理（Cleanup）

| 模块 | 说明 | 难度 |
|------|------|------|
| **智能扫描 Smart Scan** | 一键综合扫描：系统垃圾 + 隐私 + 性能问题，给出总览与一键修复。是首页主入口。 | 中 |
| **系统垃圾 System Junk** | 用户/系统缓存、日志、旧更新包、各 App 缓存、废纸篓。**（下列子项为规划项，当前版本尚未实现：语言包清理——无 `.lproj` 语言包扫描器，未枚举/裁剪各 App 多余本地化资源；损坏的登录项与偏好——无失效/损坏检测，不识别指向已不存在目标的 LaunchAgents/登录项或解析失败的偏好 plist。）** | 高 |
| **邮件附件 Mail Attachments** | 本地缓存的邮件附件与下载。 | 中 |
| **照片垃圾 Photo Junk** | 照片图库缓存、未优化副本。 | 中 |
| **iOS 文件 / 备份** | iTunes/Finder 旧 iOS 备份、固件、不再使用的设备支持文件。 | 中 |
| **废纸篓清空 Trash Bins** | 清空所有卷宗的废纸篓。 | 低 |
| **开发者垃圾 Developer Junk**（差异化卖点） | 已交付：Xcode DerivedData / iOS DeviceSupport / Archives / 旧模拟器、Homebrew/npm/pip 缓存。**（下列为规划项，当前版本尚未实现：Docker 悬空镜像检测、`node_modules` 体积聚合扫描——注：深度扫描出于提速会剪枝 `node_modules`，即跳过不深入，并非将其识别为可清理物。）** | 高 |

### B. 应用（Applications）

| 模块 | 说明 |
|------|------|
| **卸载器 Uninstaller** | 彻底卸载应用，连同 Application Support / Caches / Preferences / Containers / LaunchAgents / 登录项 / 收据一并清除。 |
| **残留清理 Leftovers** | 扫描已删除应用遗留的孤儿文件。**（未作为独立模块交付：残留清理能力已并入「系统垃圾 System Junk」的损坏登录项/偏好扫描，以及「卸载器 Uninstaller」的连带清除。）** |
| **更新器 Updater** | 检测已安装应用的可用更新（可选，Phase 2）。 |
| **重置 Reset** | 把某个应用恢复到「刚安装」状态（清空其数据但保留 App）。**（尚未实现 / 规划中，当前版本不提供。）** |

### C. 文件与空间（Files & Space）

| 模块 | 说明 |
|------|------|
| **空间透镜 Space Lens** | 全盘可视化 treemap / 旭日图，交互式钻取文件夹，直接定位空间大户。 |
| **大文件与旧文件 Large & Old Files** | 按大小 / 最后访问时间筛选，快速找出可清理的大块头。 |
| **重复文件 Duplicate Finder** | 内容级查重（含相似图片可选），智能保留建议，识别 APFS 克隆避免误判。 |
| **下载与杂物 Downloads & Clutter** | 整理下载文件夹、桌面杂物、旧安装包（.dmg/.pkg）。**（未作为独立模块交付：该能力已并入「智能扫描 Smart Scan」的残留安装包/中断下载检测，以及「大文件与旧文件 Large & Old Files」的按大小/最后访问时间筛选，故不单列为一个独立入口。）** |

### D. 性能与安全（Performance & Security）

| 模块 | 说明 |
|------|------|
| **优化 Optimization** | 管理登录项 / 启动代理 / Launch Daemons、识别高资源消耗后台进程、释放内存。 |
| **维护 Maintenance** | 已交付：清理「快速查看」缓存、清理用户字体缓存、刷新 DNS 缓存、重建「打开方式」数据库、重启访达 / 程序坞。**（下列为规划项，当前版本尚未实现：运行系统维护脚本 daily/weekly/monthly、重建 Spotlight 索引、修复磁盘权限、释放可清除空间。）** |
| **隐私 Privacy** | 清理浏览器历史/Cookie/缓存（Safari/Chrome/Firefox/Edge/Arc）、最近项目、聊天附件缓存。 |
| **威胁防护 Malware Removal**（Phase 2） | 基于特征库扫描已知 macOS 广告软件/劫持/恶意 LaunchAgent，提供移除。 |
| **菜单栏监控 Menu Bar Monitor** | 实时硬件指标 + 快捷操作。 |

## 5. 关键用户旅程

### 旅程 1：首次使用（最重要）
1. 启动 → 精致的引导欢迎页（品牌动效）。
2. 引导授予 **完全磁盘访问权限（Full Disk Access）**：清晰图文说明 + 一键跳转系统设置。
3. 安装 **特权助手**（一次性管理员授权弹窗）。
4. 进入首页 → 突出「智能扫描」大按钮。
5. 扫描中显示有质感的进度动效与实时发现项。
6. 扫描完成 → 总览卡片（可释放 X GB / 发现 Y 个问题）。
7. 用户可「一键清理」或展开逐项审阅（默认安全勾选）。
8. 清理时播放满足感动效 → 完成庆祝页（释放了多少空间）。

### 旅程 2：磁盘报满救急
搜索/直达「空间透镜」→ 可视化定位最大文件夹 → 右键在 Finder 中显示 / 直接移到废纸篓。

### 旅程 3：日常常驻
菜单栏图标常显内存/磁盘 → 卡了点一下「释放内存」→ 偶尔点「快速清理」。

## 6. 安全与信任设计（产品层）

清理类工具最大的信任风险是「误删」。产品层必须做到：
- **永远先预览，再删除**：所有待删项可展开查看路径与大小。
- **默认只勾选 100% 安全项**；有风险项（如活动中 App 缓存、体积较大的旧缓存）默认不勾并标注警告。
- **白名单 / 忽略列表**：用户可永久忽略某些路径。
- **可恢复**：优先移入「可恢复区」或废纸篓，提供「撤销上次清理」。
- **保护清单**：内置不可触碰路径（系统、SIP 保护、钥匙串、运行中应用数据）。
- **透明告知**：每一类垃圾都有「这是什么 / 删了会怎样」的说明。

## 7. 非功能性需求（NFR）

- **性能**：全盘扫描（百万级文件）应在数十秒级完成，UI 全程不卡（60/120fps）。
- **资源占用**：空闲时 CPU ≈ 0，内存占用低；菜单栏监控本身不能成为负担。
- **可靠性**：清理操作有事务性与日志，崩溃不留半删状态。
- **隐私**：本地优先，遥测匿名且可关闭；绝不上传文件内容。
- **可访问性**：VoiceOver、键盘导航、动态字体、减弱动态效果（Reduce Motion）。
- **本地化**：至少中 / 英，架构支持多语言。
- **兼容**：建议最低 macOS 13 Ventura（为用 `SMAppService`），主力适配 macOS 14/15 及最新版，Apple Silicon + Intel 通用二进制。
